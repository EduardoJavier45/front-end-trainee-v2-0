## 🎯 Objetivo del Plan Formativo

El objetivo central del curso es **desarrollar aplicaciones web desde el lado del cliente** que resuelvan necesidades organizacionales, aplicando las buenas prácticas de la industria para garantizar los niveles de calidad requeridos en el producto final.

---

## 📚 Estructura y Contenido de los Módulos

Este repositorio sigue la estructura modular del curso. Cada carpeta principal corresponde a un módulo del plan formativo.

### Módulo 1: Orientación y Metodología del Curso (M1)
- **Competencia:** Identificar las habilidades del perfil Front-End y comprender la metodología de trabajo del *bootcamp*.
- **Contenido:** Análisis del rol, contexto de la industria TI y estructura del plan formativo.

### Módulo 2: Fundamentos de la Construcción Web (M2)
- **Competencia:** Implementar la estructura, presentación y lógica inicial de una página web.
- **Tecnologías Clave:** `HTML5 Semántico`, `CSS3` (Flexbox, Grid), `JavaScript` (Básico), `Git` y `GitHub`.

### Módulo 3: Interfaces Avanzadas y Frameworks CSS (M3)
- **Competencia:** Aplicar metodologías y herramientas avanzadas para la creación de interfaces web escalables y responsivas.
- **Tecnologías Clave:** Metodologías `BEM/OOCSS`, Preprocesador `SASS`, Framework `Bootstrap 4`.

### Módulo 4: Práctica Integradora I (M4)
- **Competencia:** Integrar los conocimientos de los módulos 2 y 3 para desarrollar un proyecto web estático completo, aplicando un flujo de trabajo profesional con Git.
- **Proyecto:** Construcción de una landing page o sitio simple completamente responsivo.

### Módulo 5: Programación Avanzada con JavaScript (M5)
- **Competencia:** Profundizar en los fundamentos de JavaScript para construir aplicaciones interactivas y robustas.
- **Conceptos Clave:** `Programación Orientada a Objetos (POO)`, `ES6+`, Manipulación del `DOM`, Asincronismo (`Promises`, `Async/Await`) y consumo de APIs con `Fetch` y `Axios`.

### Módulo 6: Desarrollo de Aplicaciones con Vue.js (M6)
- **Competencia:** Implementar interfaces de usuario interactivas y reactivas utilizando un framework moderno.
- **Tecnologías Clave:** `Vue.js 3`, Componentes, Ciclo de Vida, `Vue Router` para SPA (Single Page Applications).

### Módulo 7: Vue.js Avanzado y Pruebas de Calidad (M7)
- **Competencia:** Gestionar el estado de aplicaciones complejas y asegurar la calidad del software mediante pruebas.
- **Tecnologías Clave:** Gestión de estado con `Vuex`, Pruebas Unitarias y End-to-End (`Jest`, `Cypress`).

### Módulo 8: Construcción de Portafolio Profesional (M8)
- **Competencia:** Presentar un producto digital en un portafolio personal, aplicando buenas prácticas y utilizando herramientas como GitHub Pages.
- **Contenido:** Estructuración de portafolio, revisión de proyectos y despliegue (*deployment*).

### Módulo 9: Desarrollo de Empleabilidad en la Industria Digital (M9)
- **Competencia:** Elaborar un plan de búsqueda laboral y aplicar técnicas efectivas para entrevistas.
- **Contenido:** Creación de `Currículum Vitae` adaptado a la industria TI y preparación para entrevistas técnicas y de comportamiento.

- **Numeración:** Las subcarpetas están numeradas para seguir un orden lógico de aprendizaje

## 🎯 Proyecto Destacado: M4 - Práctica Integradora

### ✨ **Proyecto Integrador I - Sitio Web Moderno y Minimalista**

El **Módulo 4** representa el primer hito importante del curso, integrando todos los conocimientos adquiridos en los módulos M2 y M3. Se ha desarrollado un sitio web completo con las siguientes características:

#### 🎨 **Características de Diseño**
- **CSS Moderno:** Variables CSS personalizadas (`--custom-properties`)
- **Responsive Design:** Enfoque mobile-first con breakpoints optimizados
- **Gradientes y Efectos:** Diseño visual moderno con sombras sutiles
- **Tipografía:** Fuente Inter de Google Fonts para legibilidad profesional
- **Animaciones:** Transiciones fluidas con `@keyframes` y `transform`

#### ♿ **Accesibilidad y UX**
- **WCAG Compliance:** Cumple estándares de accesibilidad web
- **Modo Oscuro:** Soporte automático con `prefers-color-scheme`
- **Estados de Focus:** Navegación optimizada por teclado
- **Movimiento Reducido:** Respeta `prefers-reduced-motion`

#### ⚡ **Funcionalidad JavaScript**
- **ES6+ Features:** Arrow functions, template literals, destructuring
- **Intersection Observer:** Animaciones al hacer scroll
- **Event Handling:** Interactividad moderna y eficiente
- **Console Styling:** Logging mejorado para debugging

#### 📁 **Estructura del Proyecto**
```
M4_Practica_Integradora/
├── index.html          # HTML5 semántico mejorado ✅
├── css/
│   └── styles.css      # CSS moderno y escalable ✅
├── js/
│   └── main.js         # JavaScript ES6+ interactivo ✅
└── assets/             # Recursos multimedia (preparado)
```
