Bienvenido al Ejercicio de CSS

Este es el primer párrafo. Aquí puedes practicar selectores y propiedades CSS básicas como cambiar colores, tamaños de fuente, márgenes y más.

Este es el segundo párrafo. Recuerda que CSS (Cascading Style Sheets) es el lenguaje que usamos para estilizar nuestras páginas web y hacerlas visualmente atractivas.

Conceptos CSS que aprenderás:

  • Selectores básicos (elementos, clases, IDs)
  • Propiedades de texto (color, font-size, text-align)
  • Modelo de caja (margin, padding, border)
  • Colores y fondos (color, background-color)
  • Decoración de texto (text-decoration)

Para más información sobre CSS, visita MDN Web Docs - CSS
