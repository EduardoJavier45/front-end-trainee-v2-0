1. Variables CSS (Custom Properties)

Las variables CSS permiten almacenar valores que se pueden reutilizar en todo el documento, facilitando el mantenimiento y los cambios de diseño.

Color Primario

var(--color-primary)

Color Secundario

var(--color-secondary)

Color Acento

var(--color-accent)

Cómo funcionan:

  • Definición: Se declaran con --nombre-variable en :root
  • Uso: Se acceden con var(--nombre-variable)
  • Ventaja: Cambiar un valor en un lugar afecta todo el sitio

4. Transiciones CSS

Las transiciones permiten cambiar valores de propiedades CSS de forma suave durante un período de tiempo determinado.

Hover para cambiar color

transition: background-color 0.3s;

Hover para escalar

transition: transform 0.3s;

Hover para múltiples efectos

transition: all 0.3s;

Propiedades de transición:

  • transition-property: Propiedad a animar (color, transform, all, etc.)
  • transition-duration: Duración de la transición (0.3s, 500ms)
  • transition-timing-function: Curva de aceleración (ease, linear, ease-in-out)
  • transition-delay: Retraso antes de iniciar
  • Forma corta: transition: property duration timing-function delay;

5. Animaciones CSS (@keyframes)

Las animaciones permiten crear movimientos más complejos con múltiples estados, definidos mediante @keyframes.

Bounce

Rotate

Pulse

Slide

Propiedades de animación:

  • @keyframes: Define los estados de la animación (0%, 50%, 100%)
  • animation-name: Nombre del @keyframes a usar
  • animation-duration: Duración total de la animación
  • animation-timing-function: Curva de velocidad
  • animation-iteration-count: Número de repeticiones (infinite)
  • animation-direction: Dirección (normal, reverse, alternate)

Breakpoints comunes:

  • Mobile: @media (max-width: 767px)
  • Tablet: @media (min-width: 768px) and (max-width: 1023px)
  • Desktop: @media (min-width: 1024px)
  • Mobile First: Empezar con estilos móviles y usar min-width
  • Desktop First: Empezar con desktop y usar max-width
