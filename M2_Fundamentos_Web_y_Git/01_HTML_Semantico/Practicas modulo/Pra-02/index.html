Introducción al HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza una serie de elementos y etiquetas para estructurar el contenido.

Elementos HTML Comunes

  • <h1> a <h6>: Encabezados de diferentes niveles.
  • <p>: Párrafos de texto.
  • <a>: Enlaces a otras páginas o recursos.
  • <img>: Imágenes.
  • <div>: Contenedores genéricos para agrupar contenido.
